¿Qué es Python?

Lenguaje de Programación

Python es un lenguaje de programación interpretado, de alto nivel, multiparadigma y con una sintaxis clara y legible.

⚡

Desarrollo web, ciencia de datos, inteligencia artificial, automatización, scripting, análisis de datos y más.

Características Principales

Sintaxis simple, tipado dinámico, multiplataforma, gran ecosistema de librerías y comunidad activa.

🔧 Instalación de Python

En Windows:

  1. Descarga Python desde python.org
  2. Ejecuta el instalador
  3. MARCA "Add Python to PATH"
  4. Click en Install Now

En macOS:

brew install python3

En Linux:

sudo apt-get install python3 python3-pip

Entorno virtual:

python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

📦 Conceptos Básicos

Variables y Tipos:

nombre = "Juan"           # str
edad = 30               # int
altura = 1.75           # float
es_estudiante = True    # bool
hobbies = ["leer"]      # list
persona = {"nombre": "Juan"}  # dict

Estructuras de Control:

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

for i in range(5):
    print(i)

while contador < 10:
    contador += 1

✅ Buenas Prácticas

PEP 8 - Style Guide:

  • Usa 4 espacios para indentación
  • Nombres de variables en minúsculas
  • Clases en CamelCase
  • Máximo 79 caracteres por línea

Testing con pytest:

def test_suma():
    assert suma(2, 3) == 5
